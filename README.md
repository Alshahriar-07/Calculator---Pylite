# 🧮 Six Digit Calculator

A modern and elegant calculator built with **Python** and **Tkinter**. Designed with a clean interface, smooth animations, and keyboard support while remaining lightweight and contained in a single Python file.

---

## ✨ Features

* 🎨 Modern minimal UI
* 🖱️ Smooth button press animation
* ✨ Hover effects
* ⌨️ Full keyboard support
* 🔢 12-digit display
* ➕ Basic arithmetic operations

  * Addition (`+`)
  * Subtraction (`-`)
  * Multiplication (`*`)
  * Division (`/`)
* 🧹 Clear display (`C`)
* ⚡ Fast calculation using Python
* 📄 Single-file project
* 📦 No external dependencies

## ⌨️ Keyboard Shortcuts

| Key       | Action                |
| --------- | --------------------- |
| 0-9       | Enter numbers         |
